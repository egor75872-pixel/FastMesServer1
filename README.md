# FastMes - Simple Messenger

## Quick Start

1. Install dependencies:
```
npm install
```

2. Run the server:
```
npm start
```

3. Open in browser:
```
http://localhost:3000
```

## Features
- User registration and login
- Real-time messaging
- Friend requests
- 7-digit user IDs

## Deployment on Render.com

1. Push to GitHub
2. Create Web Service on render.com
3. Build Command: `npm install`
4. Start Command: `npm start`

Done!
